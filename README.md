# SOLPICK: 일사량·발전량 예측 기반 태양광 설치 경제성 분석 및 시공사 매칭 플랫폼

<img width="769" height="314" alt="image" src="https://github.com/user-attachments/assets/b32d1645-e8d0-4792-b75a-965526040ded" />


SOLPICK은 사용자가 지역, 지붕 면적, 방향, 경사, 음영 조건, 전기요금 정보를 입력하면 예상 태양광 발전량과 경제성을 계산하고, 조건에 맞는 시공사를 비교·추천하는 웹 기반 의사결정 지원 서비스입니다.

본 저장소는 아이디어 기획안을 바탕으로 구현한 서비스 프로토타입입니다. 현재 버전은 데모용 지역별 일사량 테이블, 지원금 규칙, 시공사 샘플 데이터를 사용하며, 실제 서비스 적용 시 기상청 기상자료개방포털, 한국에너지공단, 실제 시공사 데이터베이스와 연동할 수 있도록 구조화했습니다.

## 1. 프로젝트 개요

태양광 설치 의사결정 과정에서는 일사량, 예상 발전량, 설치비, 지원금, 전기요금 절감액, 시공사 품질 정보를 각각 따로 확인해야 합니다. 이 과정은 일반 사용자와 소규모 사업자에게 높은 탐색 비용을 발생시키며, 시공사 간 비교도 견적 중심으로 이루어지는 경우가 많습니다.

SOLPICK은 이러한 문제를 다음 흐름으로 해결합니다.

```text
사용자 조건 입력
  -> 지역별 일사량 프로파일 조회
  -> 방향·경사·음영 보정
  -> 월별/연간 발전량 추정
  -> 설치비·지원금·전기요금 기반 경제성 분석
  -> 시공사별 비용·효율·신뢰도 점수화
  -> 추천 시공사 및 예상 회수기간 제시
```

## 2. 주요 기능

| 기능 | 설명 |
|---|---|
| 발전량 추정 | 지역별 월평균 일사량, 지붕 조건, 모듈 효율, PR을 반영해 월별/연간 발전량 계산 |
| 경제성 분석 | 총 설치비, 지원금, 순설치비, 연간 절감액, 회수기간, 25년 ROI 계산 |
| 지원금 매칭 | 지역, 사용자 유형, 설비용량에 맞는 지원금 규칙 적용 |
| 시공사 추천 | 설치 단가, 모듈 효율, PR, 보증기간, 평점, AS 점수, 리뷰 감성 점수를 종합해 추천 |
| 웹 대시보드 | 입력 폼, 발전량 차트, 경제성 요약, 시공사 비교 테이블 제공 |

## 3. 발전량 계산 로직

기본 계산식은 다음과 같습니다.

```text
설비용량(kWp) = 설치가능면적(m²) × 모듈효율 × 1kW/m²
월 발전량(kWh) = 월 누적 일사량(kWh/m²) × 설비용량(kWp) × PR × 방향/경사/음영/온도 보정계수
```

여기서 설치가능면적은 전체 지붕 면적에 사용 가능 비율을 적용한 값입니다. 모듈효율과 면적은 설비용량 계산에 한 번만 사용하며, 이후 발전량 계산에서 중복 곱하지 않도록 구성했습니다.

## 4. 경제성 분석 로직

경제성 분석은 다음 항목을 계산합니다.

```text
총 설치비 = 설비용량(kWp) × 업체별 kW당 설치 단가
지원금 = min(설비용량 × kW당 지원금, 지원금 상한액)
순설치비 = max(총 설치비 - 지원금, 0)
자가소비 절감액 = min(연간 발전량, 연간 사용량) × 전기요금 단가
잉여전력 수익 = max(연간 발전량 - 연간 사용량, 0) × 판매 단가
연간 순편익 = 자가소비 절감액 + 잉여전력 수익 - 유지보수비
회수기간 = 순설치비 / 연간 순편익
25년 ROI = (25년 누적 순편익 - 순설치비) / 순설치비
```

## 5. 시공사 추천 로직

시공사 추천 점수는 단순 최저가가 아니라 비용, 발전 성능, 신뢰도, 보증기간, 지역 적합성을 함께 반영합니다.

```text
추천점수 = 0.35 × 비용점수
        + 0.25 × 성능점수
        + 0.20 × 신뢰점수
        + 0.10 × 보증점수
        + 0.10 × 지역점수
```

## 8. 현재 버전의 범위

현재 구현은 공공데이터 실시간 연동 전 단계의 분석 프로토타입입니다. 데모 데이터는 서비스 흐름과 계산 로직을 검증하기 위한 샘플이며, 실제 지역별 일사량·보조금·시공사 정보와 다를 수 있습니다.

향후 확장 방향은 다음과 같습니다.

- 기상청 기상자료개방포털 API 연동
- 한국에너지공단 및 지자체 보조금 공고 자동 수집
- 실제 시공사 DB 구축
- LightGBM/XGBoost 기반 발전량 보정 모델 추가
- LSTM/Temporal CNN 기반 단기 발전량 예측 모듈 추가
- 요금제별 전기요금 계산 모듈 세분화
